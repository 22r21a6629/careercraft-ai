# CareerCraft AI 🚀

AI-powered resume and cover letter writer. Built with React + Claude AI.

## Deploy on Vercel

1. Upload this folder to GitHub
2. Go to vercel.com → Import your GitHub repo
3. Click Deploy — done!

## Before going live

Replace these placeholders in your code:
- `ca-pub-XXXXXXXXXXXXXXXX` → your Google AdSense Publisher ID
- `https://buy.stripe.com/your-link` → your Stripe checkout link
